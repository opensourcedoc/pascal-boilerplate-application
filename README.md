# Pascal Boilerplate for Application

A Free Pascal boilerplate project to build a Pascal based application.

## System Requirements

* Free Pascal

## Usage

Clone the project:

```shell
$ git clone https://github.com/opensourcedoc/pascal-boilerplate-application.git myapp
```

Move your working directory to the root of *myapp*:

```shell
$ cd myapp
```

Modify *Main.lpr* as needed. You may add or remove Pascal source files (*.pas*) as needed.

Compile the application:

```shell
$ ./build
```

Run the compiled program:

```shell
$ ./run
```

Set your own remote repository:

```shell
$ git remote set-url origin https://example.com/user/project.git
```

Push your modification to your own repo:

```shell
$ git push
```
